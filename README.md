# Bytehot Infrastructure

Extracted from the ByteHot monorepo - bytehot infrastructure.

## Installation

### Maven
```xml
<dependency>
  <groupId>org.acmsl.bytehot</groupId>
  <artifactId>bytehot-infrastructure</artifactId>
  <version>1.0.0</version>
</dependency>
```

### Gradle
```gradle
implementation 'org.acmsl.bytehot:bytehot-infrastructure:1.0.0'
```
